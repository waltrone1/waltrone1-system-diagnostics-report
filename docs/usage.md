# Usage

This document explains the basic usage of **waltrone1 System-Diagnostics-Report**.

## Purpose

waltrone1 System-Diagnostics-Report is designed to create a simple diagnostics overview of a Windows system.

It can be used to collect basic system information before troubleshooting, documentation or support requests.

## Planned Report Information

The report may include information such as:

- Computer name
- Current user
- Windows version
- System architecture
- CPU information
- RAM information
- Drive and storage information
- Network adapter information
- IP address information
- Basic system status
- Optional export information

## Basic Usage

1. Download the latest release.
2. Extract the ZIP file.
3. Start the tool.
4. Generate the diagnostics report.
5. Save or export the report if needed.

## Notes

This tool does not replace professional diagnostics software.

It is intended as a simple helper tool for quick system overview and basic documentation.
